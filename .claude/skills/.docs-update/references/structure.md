# 構成

## 単一製品

```
.docs/
├── overview.md
├── architecture.md
├── glossary.md
├── notes/
└── product/
    ├── overview.md
    ├── architecture.md
    ├── operations.md
    ├── database.md
    ├── personas.md
    ├── actors.md
    ├── relationships.md
    ├── api.md
    ├── sitemap.md
    ├── stories/
    ├── models/
    ├── services/
    ├── policies/
    ├── integrations/
    ├── pages/
    ├── features/
    └── issues/
```

## 複数製品

```
.docs/
├── overview.md
├── architecture.md
├── glossary.md
├── notes/
└── products/
    ├── <product-a>/
    └── <product-b>/
```

各製品ディレクトリは単一製品の product/ と同じ構造。
